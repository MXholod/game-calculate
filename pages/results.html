Description and step-by-step instructions.

This block displays the current level and a countdown timer.

2. Managing numbers
Initial numbers to start the game.Choose between (even/odd) or mixed.
Select a range of numbers. Possible game values are from a minimum of -9999 to a maximum of 9999. The current number level range is 0 to 0

In this block, you can specify what type of numbers will be in the game: 'mixed', 'even' or 'odd', also select a range of numbers (negative, positive) and choose which signs they should have throughout the level.

3. User's result compares with the CPU result
User's current amount
VS
CPU result

When the user has opened all the cells of the cube, calculated his own result, he can enter this result in the field and confirm it with the button. This result is compared with the CPU result, after which the level is considered passed.

5. Start / Stop / Retry the game
Game start/stop/retry all levels

There are three buttons.

- The "Play the game" button can only be clicked if the user sets the range of numbers that he wants, but no less than acceptable.

- The "Stop the game" button allows the player to interrupt the game while playing a level.

- The "Try to play all levels again" button is used to replay the game again. When the level time is up or if the game is interrupted by the user, this button activates the "Play the game" button.

Starting the game is as easy as unwrapping a candy!

1. Simply select numbers from a range randomly or drag the sliders left or right.

2. After determining the range of numbers, you can click the 'Play the game' button.

3. I wish you good luck and pleasant calculations!
